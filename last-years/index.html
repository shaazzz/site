    دوره ۲۱
    از اونجا که دوره ی ۲۱ دو بخش بود (‌ یعنی اول نقره ها ی سال قبل نبودن ، بعد که برنز ها مشخص شدن و رفتند نقره ها اومدن ) ۲ تا عکس میزاریم که همه رو شامل شه ( یکی قبل از اومدن نقره ها و یکی بعدش !‌ )



    از راست به چپ ( دقیقا از راست به چپ ، اونجاهایی که دو نفر تو ۱ طول قرار دارن مشخص کردم !‌ ) :
    سامان سامی - سعید نادری - محمد معین زراعت کار - آرمان محمدی - صدرا علیمی - محمد استادمحمدی
    فرزاد عبدالحسینی ( پایین ) - مهدی خسروی - کیوان علیزاده ( بالا ) - عرفان سلطانلو - ایمان جامی مقدم
    سینا یزدان بد - مهرداد بختیاری - مجتبی انصاری - آقای ایرد موسی ( معلم ) - آرش پور ضرابی - منوا کشیشیان
    سید حامد ولیزاده - محمد امین قیاسی - محمد حسین رضایت - دانیال مهرجردی - پویا مصدق ( پایین )
    پاشا خسروی - ایمان همایونی - هومن هاشمی ( پایین )‌ - فرنود مسعودی - نوید کرهانی
    محمد لطیفیان ( پایین ) - علیرضا مشهد الکوبه - تینا ترکمان - زینب میری - پرنیان یوسفی


    فقط اونایی رو می نویسم که تو عکس قبل نبودن :
    پوریا شریفی نژاد ( نفر چهارم از سمت راست ّ) - سعید ایلچی ( ۵ نفر بعد از پوریا )
    اقبال سرجمعی (‌ نفر بعدی سعید - لباس آبی ) - علیرضا فرهادی ( ۲ نفر بعد از اقبال - بالا )
    امیر گوهرشادی ( نفر بعدی علیرضا )
    مجید فرهادی ( نفر اول از سمت چپ ) - حامد رئوفیان مقدم ( نفر چهارم از سمت چپ - بالا )
